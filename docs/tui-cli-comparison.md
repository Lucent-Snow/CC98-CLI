## 实现计划

### 阶段一：核心读取（P0，已完成）

1. **搜索功能**
   - 按 `/` 打开搜索输入框
   - 支持帖子搜索和用户搜索
   - 搜索结果列表

2. **用户详情**
   - 点击用户名进入详情
   - 显示粉丝/关注数
   - 显示最近帖子

### 阶段二：核心写入（P0，已完成）

1. **签到**
   - "我的"页面显示签到状态
   - 按 Enter 签到

2. **收藏/关注**
   - 帖子详情按 `s` 收藏/取消收藏
   - 用户详情按 `f` 关注/取消关注
   - 版面列表按 `s` 收藏/取消收藏（待做）

### 阶段三：消息增强（P1，部分完成）

1. **通知列表**
   - 通知视图已提供系统 | @ | 回复入口
   - 消息视图合并标签页仍可作为后续体验优化

2. **发私信**
   - 用户详情按 `m` 已可发送
   - 私信对话按 `r` 回复待做

3. **点赞/踩**
   - 帖子详情按 `l` 点赞、`d` 踩

### 阶段四：高级功能（P2）

1. **发帖/回帖**
   - 版面帖子列表按 `n` 发帖
   - 帖子详情按 `r` 回帖
   - 需要实现文本输入编辑器

2. **浏览历史**
   - 已通过 "我的"/"更多"入口接入

3. **收藏夹分组**
   - 已通过收藏/"更多"入口接入，分组筛选体验仍可增强

## 快捷键规划

### 全局

| 键 | 功能 |
|----|------|
| `/` | 搜索 |
| `?` | 帮助 |
| `q` | 退出 |
| `r` | 刷新 |
| `Esc` | 返回 |

### 帖子详情

| 键 | 功能 |
|----|------|
| `s` | 收藏/取消收藏 |
| `l` | 点赞 |
| `d` | 踩 |
| `r` | 回帖（P2） |
| `j/k` | 滚动 |
| `n/Space` | 下一页 |
| `【/】` | 上/下楼层 |

### 用户详情

| 键 | 功能 |
|----|------|
| `f` | 关注/取消关注 |
| `m` | 发私信 |

### 版面列表

| 键 | 功能 |
|----|------|
| `s` | 收藏/取消收藏版面 |
| `Enter` | 进入版面 |
| `n` | 发帖（P2） |

### 私信对话

| 键 | 功能 |
|----|------|
| `r` | 回复 |
| `n/Space` | 更早消息 |
