# Changelog

## Unreleased

### 改进

- 重构 TUI：删除旧的单文件实现，拆分为入口、控制器、渲染器、组件、状态、导航和帖子阅读器模块。
- TUI 异步副作用集中到 `TuiController`，修复重构后预览、进入主题、搜索、返回和退出渲染容易断裂的问题。
- 增强 TUI 数据归一化，兼容 CC98 API 响应中的大小写字段和嵌套主题/用户数据。
- 更新 TUI 状态栏提示，按当前上下文显示更短的操作提示。

### 修复

- 修复 `mapLimit` 并发限制实现错误。
- 修复退出后异步加载仍可能继续渲染终端的问题。

## v0.2.0 - 2026-05-26

这次更新主要改进 TUI 的浏览体验，并补上本地缓存和更新检查能力。

### 新增

- 新增 `cc98 update`，可检查 GitHub Release 上的最新版本。
- 新增 `cc98 cache` 命令，支持查看缓存统计、清理过期缓存和清空缓存。
- TUI 新增设置页，支持缓存清理、快捷键帮助和检查更新。
- 帖子阅读新增楼层概念，支持数字加 Enter 跳转楼层，`【` / `】` 在楼层之间切换。
- 帖子阅读右栏新增当前楼层、当前行、图片、链接、评分和热门回复信息。

### 改进

- 优化 TUI 左栏、中栏和右栏的信息架构，每个栏目拥有独立的右侧说明。
- 优化消息、设置等栏目标题和选中态显示。
- 优化帖子内 `j/k` 行滚动，保留逐行查看内容的能力。
- 增强本地缓存，加入文件缓存统计、过期清理和内存缓存上限。

### 修复

- 修复进入设置后按键处理错误导致无法执行设置项的问题。
- 修复版面帖子列表操作菜单中的返回动作无效的问题。
- 修复 token 刷新后可能把当前账号错误保存为 `default` 的问题。

## v0.1.0 - 2026-05-26

- 初始版本，提供 CC98 CLI 和 TUI 基础浏览能力。
