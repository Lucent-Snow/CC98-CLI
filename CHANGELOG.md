# Changelog

## v0.5.1 - 2026-05-31

这次更新重整 TUI 的阅读入口和连续列表体验，并修复长帖跳转时一次性加载过多楼层的问题。

### 新增

- 新帖视图新增“最新 / 随机 / 推荐”标签；随机和推荐使用换一批模式刷新。
- 关注视图新增“版面 / 用户 / 追踪”标签，默认展示关注版面的帖子流。
- CLI 新增 `cc98 topic recommendation [--size n]`，对应随机推荐主题接口。

### 改进

- 左栏导航调整为“十大”置顶，保留独立十大入口。
- 可分页列表改为自动补页填满窗口，移除“还有 x 项”的显式剩余提示。
- 长帖跳转到远处楼层时直接请求目标窗口，不再加载所有中间楼层。
- 搜索关闭和搜索结果进入后会恢复原列表位置，刷新当前主题会保留阅读位置。

### 修复

- 隐藏账号列表中遗留的 `default` 账号项。
- 清理表情包预览中的“第 N 批”测试分组。

## v0.5.0 - 2026-05-30

这次更新集中改进 TUI 的媒体体验，并补上自动签到能力。

### 新增

- 新增终端图片预览能力，支持 iTerm2、Kitty 和 Sixel 协议。
- 帖子图片改为正文单行占位、右栏预览，避免长图撑开正文和滚动跳跃。
- 新增图片后台缓存，`o` 打开图片时优先复用缓存。
- 新增 `c` 复制图片本体到剪贴板；链接行仍复制链接 URL。
- 设置页新增“自动签到”开关，默认关闭；开启后启动 TUI 时会为所有本地账号签到一次。
- 新增终端像素表情包预览入口。

### 改进

- 扩展 UBB 图片识别，支持 `[img]`、带参数 `[img=...]`、`<img src>`、附件标签、图片链接和裸图片 URL。
- 改进 ANSI 宽度计算，正确处理 OSC/APC/DCS 图片 escape 序列。
- 减少 TUI 整屏清空造成的闪烁。
- macOS 下 `o` 打开图片会使用 Preview。

### 修复

- 修复图片内联渲染导致正文间距异常、图片过大和长图滚动体验差的问题。
- 修复图片相关快捷键提示不清晰的问题。

## v0.4.2 - 2026-05-29

这次更新修复了 v0.4.1 发布流程中的 npm 2FA 自动发布问题，并整理 0.4.x 的发布说明来源。

### 改进

- GitHub Actions 发布流程改用 npm Trusted Publishing，支持通过 OIDC 自动发布 npm 包并附带 provenance。
- GitHub Release 正文改为从 `CHANGELOG.md` 中对应版本段落生成，确保发布页内容和项目变更记录一致。

### 发布说明

- 推送 `v0.4.2` tag 后会自动执行类型检查、npm 发布和 GitHub Release 创建。
- 需要在 npm 包设置中将 `Lucent-Snow/CC98-CLI` 的 `release.yml` 配置为 Trusted Publisher。

## v0.4.0 - 2026-05-28

这次更新增强了 WebVPN 的可靠性和会话持久化，并改进了 TUI 的帖子阅读体验。

### 新增

- WebVPN 会话持久化：支持从 `vpn.json` 加载已登录的 Cookie，无需每次重新登录。
- WebVPN CLI 启动集成：CLI 命令自动检测网络环境并初始化 WebVPN。
- TUI 帖子楼层跳转：支持 `gg` 跳转到顶部，`G` 跳转到底部。

### 改进

- WebVPN 重定向处理：支持相对 URL 解析、重试限制（最多 10 次）和 301/302/303 方法转换。
- WebVPN Cookie 解析：改进 `Set-Cookie` 头解析，正确处理包含逗号的值。
- WebVPN 错误提示：未登录时给出明确提示，引导用户运行 `cc98 vpn login`。
- TUI UBB 渲染：增强链接和图片的 ANSI 样式。
- 重构 TUI：删除旧的单文件实现，拆分为入口、控制器、渲染器、组件、状态、导航和帖子阅读器模块。
- TUI 异步副作用集中到 `TuiController`，修复重构后预览、进入主题、搜索、返回和退出渲染容易断裂的问题。
- 增强 TUI 数据归一化，兼容 CC98 API 响应中的大小写字段和嵌套主题/用户数据。
- 更新 TUI 状态栏提示，按当前上下文显示更短的操作提示。

### 修复

- 修复 `mapLimit` 并发限制实现错误。
- 修复退出后异步加载仍可能继续渲染终端的问题。
- 修复 WebVPN 网络检测不应发送 Cookie 到外部服务。

## v0.2.0 - 2026-05-26

这次更新主要改进 TUI 的浏览体验，并补上本地缓存和更新检查能力。

### 新增

- 新增 `cc98 update`，可检查 GitHub Release 上的最新版本。
- 新增 `cc98 cache` 命令，支持查看缓存统计、清理过期缓存和清空缓存。
- TUI 新增设置页，支持缓存清理、快捷键帮助和检查更新。
- 帖子阅读新增楼层概念，支持数字加 Enter 跳转楼层，`【` / `】` 在楼层之间切换。
- 帖子阅读右栏新增当前楼层、当前行、图片、链接、评分和热门回复信息。

### 改进

- 优化 TUI 左栏、中栏和右栏的信息架构，每个栏目拥有独立的右侧说明。
- 优化消息、设置等栏目标题和选中态显示。
- 优化帖子内 `j/k` 行滚动，保留逐行查看内容的能力。
- 增强本地缓存，加入文件缓存统计、过期清理和内存缓存上限。

### 修复

- 修复进入设置后按键处理错误导致无法执行设置项的问题。
- 修复版面帖子列表操作菜单中的返回动作无效的问题。
- 修复 token 刷新后可能把当前账号错误保存为 `default` 的问题。

## v0.1.0 - 2026-05-26

- 初始版本，提供 CC98 CLI 和 TUI 基础浏览能力。
